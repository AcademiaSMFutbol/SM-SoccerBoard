# SM SoccerBoard Pro v70

Pizarra táctica profesional para entrenadores de fútbol base.

## Archivos

```
www/
├── index.html       ← HTML principal
├── style.css        ← Estilos
├── script.js        ← Lógica completa
├── campoentero.png  ← Fondo campo completo (mantener del proyecto original)
├── mediocampo.png   ← Fondo medio campo
├── campoejercicio.png
└── futsal.png       ← (opcional)
```

## Funcionalidades v70

- ✅ 4 equipos con color principal + color de rayas (equipaje a rayas)
- ✅ Material: balón, cono alto, chincheta (cono bajo), pica, valla, escalera, peso
- ✅ Líneas rectas, curvas y polígonos por taps (zigzag/cambios dirección)
- ✅ Todas las líneas con flecha activable/desactivable + estilo punteado
- ✅ Animación fluida por pasos con easing y control de velocidad (slider)
- ✅ Exportación PNG (alta resolución)
- ✅ Exportación WebM/VP8 (PC y Android)
- ✅ Exportación GIF animado (iOS y universal)
- ✅ Biblioteca de 6 tareas predefinidas en JSON
- ✅ Inspector por elemento: color, número, rayas, flecha, punteado
- ✅ Undo ilimitado (40 estados)
- ✅ Pellizco para escalar elementos
- ✅ Tap para rotar elementos 15°
- ✅ Selector de campo (completo, medio, ejercicio, futsal, sin líneas)
